# Real-Time Log Intelligence Platform

A production-grade real-time log intelligence platform that ingests logs using Apache Kafka, detects anomalies using statistical methods and LLMs, and provides real-time alerting and observability dashboards.

---

## Features

- Real-time Kafka log ingestion
- Statistical anomaly detection
- Z-score based outlier detection
- LLM-powered anomaly explanations
- Elasticsearch log storage
- Slack alerting pipeline
- Real-time dashboard
- Searchable log explorer

---
