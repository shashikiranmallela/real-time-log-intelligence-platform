# Day 4.1 - Alert Severity Colors

## Objective

Improve alert visibility by assigning different colors based on alert severity.

## Features Implemented

- HIGH alerts displayed in dark red
- MEDIUM alerts displayed in dark yellow
- LOW alerts displayed in dark green
- Dynamic color selection using helper function

## Files Modified

- frontend/src/components/AlertPanel.jsx

## Outcome

Users can quickly identify critical alerts from the dashboard without reading every alert message.
