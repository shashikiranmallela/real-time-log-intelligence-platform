# Day 3.2 - Error Level Distribution Chart

## Objective

Visualize log severity distribution.

## Levels

- INFO
- WARN
- ERROR

## Implementation

Created ErrorLevelChart.jsx component.

Fetched recent logs from backend.

Calculated frequency of:

- INFO logs
- WARN logs
- ERROR logs

Displayed results using Bar Chart.

## Outcome

Provides quick operational visibility into platform health.

Example:

INFO : 5
WARN : 3
ERROR : 2
