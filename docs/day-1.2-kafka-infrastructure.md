# Day 1.2 — Kafka Infrastructure Setup

## Objectives

- Setup Apache Kafka locally using Docker
- Configure Kafka broker and Zookeeper
- Prepare infrastructure for real-time log ingestion

---

## Components

### Zookeeper
Used for Kafka coordination and metadata management.

### Kafka Broker
Handles real-time message streaming.

---

## Ports

- Kafka: 9092
- Zookeeper: 2181

---

## Notes

Kafka is the backbone of the real-time streaming architecture for this platform.
